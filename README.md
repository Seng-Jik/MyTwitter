# MyTwitter

## 服务器启动方式

在`MyTwitter.Server`目录下执行`dotnet run -c Release`即可。

## 客户端启动方式

在`MyTwitter.ClientCUI`目录下执行`dotnet run -c Release`即可。

该程序内已经自带操作说明。

## 性能测试模拟器启动方式

在`MyTwitter.Simulator`目录下执行`dotnet run -c Release`即可。

模拟器中的用户名已经添加了PID，可以同时启动多个实例。
